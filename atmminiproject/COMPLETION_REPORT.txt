================================================================================
                      PROGRAMMING CONCEPTS USED
================================================================================

1. PROCEDURAL PROGRAMMING
   - Pure function-based architecture
   - No class definitions
   - Linear control flow
   - Modular function organization

2. NAMESPACE ORGANIZATION
   - CommonUser namespace (user operations)
   - AdminUser namespace (admin operations)
   - FileHandler namespace (file I/O operations)

3. FILE HANDLING
   - Data persistence in plain text files
   - Pipe-delimited format for easy parsing
   - Real-time updates
   - Transaction logging with timestamps

4. DATA STRUCTURES
   - Account struct: (accountNumber, PIN, balance, name)
   - AdminAccount struct: (adminID, password, name)

================================================================================
                          DATA FILE FORMATS
================================================================================

users.txt (Pipe-delimited):
Format: accountNumber|PIN|balance|name
Example: 1001|1234|50000|Ansu Kumar

transactions.txt (Pipe-delimited):
Format: accountNumber|type|amount|timestamp
Example: 1001|WITHDRAW|5000|2024-05-26 14:30:45

admin.txt (Pipe-delimited):
Format: adminID|password|name
Example: 101|admin123|System Administrator

blocked.txt (One per line):
Format: accountNumber
Example: 1005
